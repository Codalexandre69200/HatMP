4) ## Installation pour le développement / Installation for development

J'utilise Tauri pour dévlopper ce jeux. Il prend en charge Windows, Android, Mac et Linux.
Les outils nécessaires sont indiqué dans le site web de Tauri.
Pour la prise en charge d'Android, il faut installer java (JDK21), la dernière version de Gradle et Android Studio.
J'utilise PNPM comme gestionnaire de paquet car moins lourd que npm du fait de la centralisation des paquets.
Pour gérer la sécurité du projects j'utilise SonarQube Cloud et semgrep.

I use Tauri to develop this game. It supports Windows, Android, Mac, and Linux.
The necessary tools are indicated in the website of Tauri.
To support Android, you need to install java (JDK21), the latest version of Gradle and Android Studio.
I use PNPM as a package manager because it is less cumbersome than npm due to the centralization of packages.
To manage the security of the projects, I use SonarQube Cloud and semgrep.
